## Getting started

### Installation

If you want to use this library inside your project, install it with npm:

```sh
npm i mobrix-engine
```

<br>

---

## Tests

Unit tests for every lib function are located inside `tests` folder. The test script is executed with pre-defined test command:

```sh
npm run test
```
